System Requirements

You must have LabVIEW 7.0 or later and NI-DAQ 7.3 or later to use NI-ELVIS 2.0 or later.

What's New?

NI-ELVIS 2.0.1 includes the following new feature:

  • Added support for National Instruments M Series devices

NI-ELVIS 2.0 includes the following new features:

  • Addition of a calibration utility for the variable power supplies and the function generator DC offset
  • Addition of support for software nulling with the NI ELVIS - Digital Multimeter Soft Front Panel (SFP) instrument
  • Improved NI ELVIS - Oscilloscope SFP scan rate that allows scanning at the max rate supported by the data acquisition (DAQ) device connected to the NI ELVIS hardware.
  • Added support for NI ELVIS - Oscilloscope SFP autoscaling
  • Added software support for NI-DAQmx, the robust, sophisticated new driver for National Instruments DAQ devices
  • Removed support for Traditional NI-DAQ
  • The Device Number parameter has been changed to a I/O name control type named Device Name.

Note  NI-ELVIS 2.0 modifies the NI-ELVIS 1.0 application programming interface (API). Programs written using NI-ELVIS 1.0 must be modified to work with NI-ELVIS 2.0.

Installation

For Windows NT 4.0 users, please install Service Pack 6 and Internet Explorer 5.0 or later before installing NI ELVIS. Refer to the Where to Start with NI ELVIS for installation instructions. You can access this document by selecting Start»Programs»National Instruments»NI ELVIS 2.0»Where to Start with NI ELVIS from the Start menu.

Documentation

The following list describes the NI ELVIS documentation that should be installed with NI-ELVIS.

  • Where to Start with NI ELVIS - This document includes important installation and configuration instructions you should read before using NI ELVIS.
  • NI ELVIS User Manual - This document provides more detailed information about the NI ELVIS hardware and software, including specifications and safety information.

To download the latest NI ELVIS documentation or to search for documentation for your DAQ device, refer to ni.com/manuals.

Known Issues

If you have LabVIEW 7.0 and LabVIEW 7.1 both installed on your computer, the NI-ELVIS icons will only appear in the functions palette (at Instrument I/O»Instrument Drivers»NI ELVIS) in the LabVIEW 7.1 environment. To access the VIs in the LabVIEW 7.0 environment, navigate to <LabVIEW 7.1>\instr.lib\NI ELVIS, which is the default installation location, or to the alternate location to which you installed your NI ELVIS software.

Note  If you save NI-ELVIS VIs in LabVIEW 7.1, you will be unable to reopen them in LabVIEW 7.0.
