# Lab Codebase
Code for the John Markert lab at UT Austin.

# MRFM Project
	Covers programs used with the NMRFM systems.
	# Amplitude Spectrum
	Programs for pulling the data from the amplitude spectrum of noise.
	# archive - deprecated Peter programs
	half-finished, broken, and unused programs from Peter Kampschroeder
	# archive - Manzaneras programs
	programs of uncertain use from Isaac Manzaneras's time.
	# Keithley 27XX
	Driver utilities for the Keithley 27XX series digital multimeter
	# utilities
	Programs for debugging or for use as sub-vis in the main programs
